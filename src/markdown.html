SPSSINC RANPRED Extension Command

Calculate predictions from a random forest object in memory or from a saved R workspace. 'Random Forest' is a trademark of Breiman and Cutler.

SPSSINC RANPRED FOREST=“filespec” PREDTYPE=RESPONSE** or PROBABILITIES or VOTES CHECKMISSING=NO** or YES
ID=varname

/SAVE PREDVALUES=datasetname

/PRINT PREDICTIONS

* Required ** Default

SPSSINC RANPRED /HELP. prints this information and does nothing else.

SPSSINC RANPRED.

Split files and weight are not honored by this command.

FOREST can be specified to name an R workspace saved by SPSSINC RANFOR containing the estimation information needed for prediction. By default, the workspace is expected to already be loaded. This will be the case if RETAINFOREST was specified in that command.

Categorical variables in the prediction dataset, as determined by the estimation dataset,
are automatically converted to factors using the values found in the estimation dataset.
Category values for the prediction dataset must all exist in the estimation dataset.

CHECKMISSING. Missing predictor values are not permitted when predicting. Any missing values will cause the prediction process to fail for the entire dataset. Use CHECKMISSING=YES to check for and remove any cases with missing values from the prediction dataset. An ID variable is required if CHECKMISSING=YES.

ID can specify the name of a variable in the input dataset to be written to the prediction dataset as an id variable and used if predictions are displayed.

PREDTYPE specifies the type of prediction. The choices are

  • RESPONSE: the predicted response is computed.
  • PROBABILITIES: the response probability is computed for each category
  • VOTES: the number of votes is computed for each category.

Probabilities and votes can only be specified for classification trees.

SAVE

PREDVALUES can be specified to name a dataset to hold the predicted values. The name must not already be in use.

PRINT

PREDICTIONS produces a pivot table listing the predictions.

Either or both of PREDVALUES or PRINT PREDICTIONS must be specified.

This extension command requires the R programmability plug-in and the R randomForest package.
